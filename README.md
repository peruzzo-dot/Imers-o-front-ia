📺 Netflix - Imersão Front-End Alura
Este projeto foi desenvolvido durante a Imersão Front-End promovida pela Alura. O objetivo foi criar uma interface inspirada na Netflix, focando em práticas modernas de desenvolvimento web, semântica HTML e responsividade.

🚀 Tecnologias Utilizadas
HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização, utilização de Flexbox e Grid para o layout.

JavaScript: Adição de comportamentos dinâmicos (se aplicável ao seu código atual).

GitHub Pages: Utilizado para o deploy e hospedagem do projeto.

🧠 O que eu aprendi
Durante o desenvolvimento deste projeto, foquei em conceitos fundamentais para o desenvolvimento web:

Semântica: Organização de tags para melhor acessibilidade e SEO.

Responsividade: Ajuste do layout para que a interface seja visualmente agradável em diferentes tamanhos de tela.

Metodologia BEM (Block Element Modifier): Organização das classes CSS para manter o código limpo e escalável.

Versionamento: Prática de comandos Git e gestão de repositório no GitHub.
